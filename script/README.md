## Code For EarlyMix.

EarlyMix is an advanced method for Early Time Series Classification (ETSC), designed to tackle the challenges posed by data insufficiency in early segments of time series. The method introduces a hierarchical mixing strategy consisting of Sample Mixing and Latent Mixing to enhance the model's feature representation and improve prediction accuracy. 

Download the Datasets: https://www.timeseriesclassification.com/dataset.php

Environment: ``` pip install -r requirements.txt ```


you can run : ``` sh run_earlyts_all.sh ```

All parameters are set to the default values of argparse, and users can modify them as needed through the command line.
